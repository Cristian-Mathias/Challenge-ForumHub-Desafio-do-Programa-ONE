# FórumHub API

Projeto desenvolvido como parte do Challenge Back End da Alura.

A aplicação consiste em uma API REST para gerenciamento de tópicos de um fórum, permitindo:
- Criar um novo tópico
- Listar todos os tópicos
- Consultar um tópico específico
- Atualizar um tópico
- Excluir um tópico

O projeto utiliza Spring Boot, segue boas práticas REST, possui validações de regras de negócio,
persistência em banco de dados relacional e autenticação/autorização.
